# Dataview Starter Snippets

These files are not magic. They are just practical starting points for the two questions that tend to matter quickly:

- which Lernfelder did I neglect this month
- how much report / learning time is still missing

Copy them into your vault and adapt the folder names and field names to your setup.

Included:

- [monthly-learning-gaps.md](monthly-learning-gaps.md)
- [reporting-hours.md](reporting-hours.md)

They assume the user already installed `Dataview`.
