# Spaced Repetition Engine

This plugin is still marked experimental, but the direction is simple: if your vault already knows what you studied and when you last touched it, it should be able to tell you what deserves another pass today.

The current version scans due reviews, calculates the next interval from a simple rating, and writes review queues into a `Periodic Notes`-friendly folder structure. It is intentionally understandable before it becomes clever.
