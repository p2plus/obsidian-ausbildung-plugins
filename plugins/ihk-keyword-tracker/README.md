# IHK Keyword Tracker

This plugin is for the uncomfortable but important question: are the right topics actually present in the vault, or do they only feel present because the folder tree looks full?

It scans configured IHK-relevant keywords across your notes and writes a small coverage report. That makes it easier to notice blind spots before they become exam problems. The current version is intentionally simple and transparent instead of pretending to understand semantics it cannot justify yet.
