# Quiz Generator aus Markdown

Sometimes the fastest way to test yourself is not to search for a perfect question bank but to turn your own notes into one. That is what this plugin is for.

It reads the current markdown note, extracts plausible prompts from headings and bullet structure, and writes a quiz file that the exam simulator can already use. No AI is required for the basic flow, and the output stays clean enough for `Metadata Menu` and plain YAML workflows.
